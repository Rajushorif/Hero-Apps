# Hero.IO

Hero.IO is a responsive demo App Store built with React and Vite. It showcases a catalog of mobile apps, supports searching, sorting, viewing details with review charts, and installing apps locally via localStorage.

## Description

This project implements the assignment requirements: a header with navigation, a footer, a home page with banner and top apps, an all-apps page with live search and sorting, app details with a review chart (Recharts), and an installation page that persists installed apps in `localStorage`.

## Technologies

- React 19
- Vite
- react-router-dom (routing)
- Recharts (charts)
- Plain CSS for responsive layout
